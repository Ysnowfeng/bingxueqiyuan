# 小增老师个人网站 - 操作手册

## 🚀 部署指南

### 方式一：Vercel 部署（推荐，免费）

1. **创建 GitHub 仓库**
   ```bash
   cd personal-site
   git init
   git add .
   git commit -m "Initial commit"
   ```
   
   然后在 GitHub 创建新仓库，推送代码：
   ```bash
   git remote add origin https://github.com/你的用户名/personal-site.git
   git push -u origin main
   ```

2. **连接 Vercel**
   - 访问 [vercel.com](https://vercel.com)
   - 使用 GitHub 账号登录
   - 点击 "Add New Project"
   - 选择你的 GitHub 仓库
   - 点击 "Deploy"

3. **访问你的网站**
   - 部署完成后，Vercel 会给你一个 URL：`https://your-project.vercel.app`
   - 可以自定义域名

### 方式二：本地预览

```bash
cd personal-site
npm install
npm run dev
```
然后访问 http://localhost:3000

## 📝 日常维护

### 1. 修改个人信息

**修改网站标题和描述** - 编辑 `src/app/layout.tsx`：
```typescript
export const metadata: Metadata = {
  title: "你的名字 | 全栈开发者",  // 改这里
  description: "你的个人简介...",    // 改这里
};
```

### 2. 修改首页内容

编辑 `src/app/page.tsx`：
- Hero 区域：头像、标题、简介
- 技能标签：修改 skills 数组
- 最新文章：修改 posts 数组

### 3. 更新博客文章

编辑 `src/app/blog/page.tsx` 中的 posts 数组，添加新文章：
```typescript
{
  slug: "my-new-post",           // URL 路径
  title: "新文章标题",            // 文章标题
  date: "2026-03-23",            // 发布日期
  excerpt: "文章摘要...",         // 摘要
  tags: ["标签1", "标签2"],       // 标签
  readingTime: "5 min",          // 阅读时间
},
```

### 4. 更新项目展示

编辑 `src/app/projects/page.tsx` 中的 projects 数组：
```typescript
{
  name: "项目名",                 // 项目名称
  description: "项目描述",         // 项目描述
  tech: ["React", "Node.js"],     // 技术栈

```

### 5. 修改联系方式

编辑 `src/app/contact/page.tsx` 中的联系方式。

## 🌐 域名绑定

### Vercel 域名设置
1. 进入 Vercel 项目 Dashboard
2. 点击 "Settings" → "Domains"
3. 输入你的域名（如 `xiaozeng.com`）
4. 按要求添加 DNS 记录

### 阿里云/腾讯云 DNS 配置
添加以下记录：
- 类型：A
  - 主机记录：@
  - 记录值：76.76.21.21

- 类型：CNAME
  - 主机记录：www
  - 记录值：cname.vercel-dns.com

## 🔧 高级配置

### 接入真实联系表单

编辑 `src/app/api/contact/route.ts`，接入邮件服务：

**使用 Resend（推荐）**
```bash
npm install resend
```

```typescript
import { Resend } from 'resend';

const resend = new Resend(process.env.RESEND_API_KEY);

export async function POST(request: Request) {
  const { name, email, message } = await request.json();
  
  await resend.emails.send({
    from: '你的网站 <noreply@你的域名.com>',
    to: ['your-email@gmail.com'],
    subject: `来自 ${name} 的留言`,
    html: `<p><b>邮箱：</b>${email}</p><p>${message}</p>`
  });
  
  return Response.json({ success: true });
}
```

然后在 Vercel 环境变量中添加 `RESEND_API_KEY`。

### 添加博客文章详情页

创建 `src/app/blog/[slug]/page.tsx`：
```typescript
import { posts } from "../../page";

export default function BlogPost({ params }: { params: { slug: string } }) {
  const post = posts.find(p => p.slug === params.slug);
  if (!post) return <div>文章不存在</div>;
  
  return (
    <div className="max-w-3xl mx-auto py-24 px-4">
      <h1>{post.title}</h1>
      {/* 添加文章内容 */}
    </div>
  );
}
```

## ❓ 常见问题

**Q: 部署后样式不对？**
A: 清理浏览器缓存，或检查 Tailwind CSS 配置。

**Q: 联系方式表单不工作？**
A: 需要接入真实的邮件服务，参考上面的高级配置。

**Q: 如何更新网站内容？**
A: 修改代码后推送到 GitHub，Vercel 会自动重新部署。
